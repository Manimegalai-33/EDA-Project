Exploratory Data Analysis (EDA) Project

Objective

To analyze the Iris dataset and discover patterns, trends, and relationships using statistical analysis and data visualization techniques.

Tools Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

Dataset

Iris Dataset

Analysis Performed

- Data Inspection
- Statistical Summary
- Missing Value Analysis
- Histogram Visualization
- Scatter Plot Analysis
- Correlation Heatmap

Key Findings

- The dataset contains 150 records and 5 columns.
- No missing values were found.
- Petal length and petal width show a strong positive correlation.
- Iris species can be differentiated using petal measurements.

Conclusion

This project demonstrates the use of Exploratory Data Analysis (EDA) techniques to understand data, identify relationships between variables, and visualize patterns effectively.
