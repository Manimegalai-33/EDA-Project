Exploratory Data Analysis Report

Dataset

Iris Dataset

Tools Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

Analysis Performed

- Data inspection
- Statistical summary
- Missing value check
- Histogram visualization
- Scatter plot visualization
- Correlation heatmap

Findings

- Dataset contains 150 records and 5 columns.
- No missing values were found.
- Petal length and petal width have a strong positive correlation.
- Iris species can be distinguished using petal measurements.

Conclusion

The project successfully identified patterns and relationships within the Iris dataset using Exploratory Data Analysis techniques.
